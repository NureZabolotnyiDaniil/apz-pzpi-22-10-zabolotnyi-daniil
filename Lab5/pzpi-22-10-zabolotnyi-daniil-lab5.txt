 5.1 Опис проекту
Smart Lighting System - це комплексна IoT-система для інтелектуального управління вуличним освітленням парків. Система складається з трьох основних компонентів: Backend (FastAPI + PostgreSQL) як центральний сервер для обробки даних, управління пристроями та надання API, Frontend (React + TypeScript) як веб-додаток для адміністрування системи з інтерактивними картами та аналітикою, та Mobile (Android + Kotlin) як мобільний додаток для оперативного звітування та моніторингу.
Система забезпечує автоматичне регулювання яскравості освітлення на основі датчиків руху, моніторинг стану обладнання, систему сповіщень про несправності та детальну аналітику використання. Головна мета полягає в економії енергії до 50% та підвищенні безпеки у парках через розумне управління освітленням.

5.2 Заплановані та виконані функції
5.2.1 Заплановані функції (Scope of Initial Release)
Backend частина системи повністю реалізована з використанням FastAPI фреймворку та включає повний CRUD функціонал для всіх основних сутностей системи, включаючи ліхтарі, парки, поломки та ремонтні роботи. RESTful API з автоматично генерованою документацією забезпечує зручну взаємодію між компонентами системи. Впроваджено надійну JWT авторизацію та систему безпеки. Система повідомлень користувачів дозволяє ефективно обмінюватися інформацією між персоналом та адміністраторами.

Frontend додаток створений на React з TypeScript та представляє собою сучасну адміністративну панель з інтуїтивним інтерфейсом. Інтерактивна карта з візуалізацією всіх ліхтарів надає наочне уявлення про стан системи в реальному часі. Система управління користувачами забезпечує гнучкий розподіл ролей та дозволів. Статистичний дашборд представляє ключові показники роботи системи у зручному для аналізу форматі. Повна багатомовність з підтримкою української та англійської мов робить систему доступною для міжнародного використання.
Мобільний додаток для Android розроблений з використанням Kotlin та сучасних архітектурних підходів Clean Architecture та MVI. Форма швидкого звітування дозволяє оперативно повідомляти про проблеми з прикріпленням фотографій та геолокації. Інтеграція з камерою та системою геолокації робить процес звітування максимально зручним та інформативним.

5.2.2 Можливий розвиток програмного забезпечення (Scope of Subsequent Releases)
IoT інтеграція вже повністю реалізована через готові API endpoints для взаємодії з ESP32 контролерами та різноманітними датчиками. Система підтримує підключення датчиків руху та струму для моніторингу стану ліхтарів та активності в парку. Телеметрія в реальному часі дозволяє миттєво реагувати на зміни в роботі обладнання. Автоматичне управління яскравістю на основі даних з датчиків забезпечує оптимальне енергоспоживання та адаптацію до поточних потреб освітлення.
Розширений функціонал включає інтеграцію з Firebase Cloud Messaging для push-сповіщень, що гарантує миттєве інформування персоналу про критичні події. Офлайн режим мобільного додатку з локальним збереженням даних та подальшою синхронізацією забезпечить безперервну роботу навіть при нестабільному інтернет-з'єднанні. Розширена аналітика та звітність на базі зібраних IoT даних допоможе приймати обґрунтовані рішення щодо оптимізації роботи системи та планування технічного обслуговування.
Покращення користувацького досвіду включає впровадження темної теми для всіх компонентів системи, покращену систему фільтрації та пошуку, можливість експорту даних у різних форматах для подальшого аналізу. Розширені налаштування сповіщень дозволять користувачам налаштувати персоналізовані правила отримання повідомлень відповідно до своїх робочих потреб та специфіки обслуговуваних територій.
